# pokedex-desktop-app-python
Pokédex interativa desenvolvida em Python com Tkinter para demonstrar consumo de API REST. A aplicação integra a PokéAPI para buscar dados em tempo real, permitindo capturar Pokémon aleatórios, exibir imagem, tipo, altura e peso, além de sistema de raridade e coleção com busca. Projeto focado em aprendizado prático de APIs e interface gráfica.
